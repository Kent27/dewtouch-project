# Getting Started

## Setup

### **1. Clone the repository**

```bash
cd root/var/www/html
git clone https://github.com/Kent27/dewtouch-project.git
cd dewtouch-project
```

### **2. Database**

Import the database from sql/db.sqp
Set up the proper db in app/Config/database.php
